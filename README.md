## Getting Started

### Prerequisites

- Node.js
- NPM
- Expo

### Installation

1. Clone the repo

```sh
git clone https://github.com/RenatoBrunetti/xfood-reactnative-expo.git
```

2. Install Packages

```sh
npm install
```

## Build and Start

1- Create an environment file (**.env**) in the root folder:

```sh
# Put your phone number here
EXPO_PUBLIC_PHONE_NUMBER=
```

2- Start Project

```sh
npx expo start
```
